# mesto-react
Мини-клон Instagram с возможностью:
* создавать/удалять фото-карточки с именем; 
* осматривать фото-карточки в большем размере; 
* редактировать информацию профиля и аватар; 
* ставить/удалять лайки на фото-карточки.
